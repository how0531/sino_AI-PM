資安與防護控管

  • 身分驗證與單一登入

    整合 Azure AD 進行單一登入，嚴格限制僅限數位金融處授權員工存取。

  • 傳輸加密與去識別化

    所有外部與內部 API 呼叫均強制採用 TLS 1.3 加密，敏感數據在查詢前均進行遮罩與去識別化處理。

  • 資料防洩漏 (DLP)

    限制資料僅能在受信任的企業 API 與 Power Automate 節點間傳遞，禁止流向非授權外部服務。
